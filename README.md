Course Completion Project

This project consists of a functional website created to facilitate the adoption of animals in need, supporting a fictional NGO called Achados e Amados.

The project includes a website developed using HTML, CSS, JavaScript, Bootstrap, and Cloudinary. The link above directs to a static demonstration version of the site.

The complete project also includes:

° an API

° a desktop application developed in C# with Windows Forms


The goal of this project is to provide a platform that helps connect animals in need with potential adopters, making the adoption process more accessible and organized. 🐾

Este projeto consiste no desenvolvimento de um site funcional voltado para facilitar a adoção de animais em situação de abandono, apoiando uma ONG fictícia chamada Achados e Amados.

O site foi desenvolvido utilizando HTML, CSS, JavaScript, Bootstrap e Cloudinary. O link acima direciona para uma versão estática de demonstração do projeto.

O projeto completo também inclui:

° uma API

° um aplicativo desktop desenvolvido em C# com Windows Forms



O objetivo do sistema é criar uma plataforma que ajude a conectar animais que precisam de um lar com possíveis adotantes, tornando o processo de adoção mais acessível e organizado. 🐾
